Задание 1 ЕГЭ

Параллельные процессы

Условие задачи

В файле информация о вычислительных процессах. Процессы могут выполняться параллельно или последовательно.

Процесс B зависит от процесса A, если для B нужны результаты A. Тогда они выполняются только последовательно.

В таблице: ID процесса, время выполнения (мс), ID процессов, от которых зависит (через ;). Если независимый — указано 0.

IDВремяЗависит от
140
230
311; 2
473

Найдите минимальное время выполнения всех процессов.

Файл с данными

Скачайте файл для решения:

data.ods

Проверка ответа

Введите ответ (в мс):

Решение

Процессы 1,2,9,10 независимы: 4,3,7,8 мс

Процесс 3 после 1 и 2: max(4,3)+1=5 мс

Процесс 4: 5+7=12 мс

Процесс 5: 5+6=11 мс

Процесс 6: 11+3=14 мс

Процесс 7: max(12,14)+1=15 мс

Процесс 8: 15+2=17 мс

Процесс 11: 7+6=13 мс

Процесс 12: 8+6=14 мс

Ответ: 17 мс
