Условие задачи

В файле содержится информация о совокупности N вычислительных процессов, которые могут выполняться параллельно или последовательно. Будем говорить, что процесс B зависит от процесса A, если для выполнения процесса B необходимы результаты выполнения процесса A. В этом случае процессы могут выполняться только последовательно.

Информация о процессах представлена в файле в виде таблицы. В первом столбце таблицы указан идентификатор процесса (ID), во втором столбце таблицы — время его выполнения в миллисекундах, в третьем столбце перечислены с разделителем «;» ID процессов, от которых зависит данный процесс. Если процесс является независимым, то в таблице указано значение 0.

Пример данных из файла:

ID процесса Время выполнения (мс) ID процесса(ов)
140
230
311; 2
473

Определите минимальное время, через которое завершится выполнение всей совокупности процессов, при условии, что все независимые друг от друга процессы могут выполняться параллельно.

Решение задачи

Пошаговое решение:

  1. Процессы с ID «1», «2», «9» и «10» независимые, поэтому их выполнение закончится на 4, 3, 7 и 8 мс соответственно.
  2. Процесс с ID «3» может выполняться только после завершения процессов с ID «1» и «2», поэтому он может завершиться на 5 мс (max(4, 3) + 1 = 5).
  3. Процессы с ID «4» и «5» зависят от процесса с ID «3», значит, они завершатся через 5 + 7 = 12 мс и 5 + 6 = 11 мс соответственно.
  4. Процесс с ID «6» зависит от процесса с ID «5», значит, он завершится через 11 + 3 = 14 мс.
  5. Процесс с ID «7» зависит от процессов с ID «4» и «6», следовательно, поскольку процесс с ID «6» завершится только на 14 мс, процесс с ID «7» выполнится на 14 + 1 = 15 мс.
  6. Процесс с ID «8» зависит от процесса с ID «7», значит, он выполнится на 15 + 2 = 17 мс.
  7. Процесс с ID «11» зависит от процесса с ID «9», поэтому он выполнится на 7 + 6 = 13 мс.
  8. Процесс с ID «12» зависит от процесса с ID «10», поэтому он выполнится на 8 + 6 = 14 мс.

Таким образом, вся совокупность процессов завершится на 17 мс.

Решение на Python

f = open("22.txt").read().split("\n")
m = {}
for i in f:
    s = i.split()
    if s[2] == '0':
        m[s[0]] = int(s[1])
    else:
        w = s[2].split(";")
        sp = []
        for j in w:
            j = j.replace(" ", "")
            sp.append(int(m[j]))
        m[s[0]] = int(s[1]) + max(sp)
print(max(m.values()))
